# React Code Standards Plugin

TypeScript/React code standards enforcement with automated review, generation, and refactoring capabilities.

## Quick Start

### Agent

**`@react-code-std`** - TypeScript/React standards baseline

### Commands

| Command      | Description                                         | Usage                                         |
| ------------ | --------------------------------------------------- | --------------------------------------------- |
| `/generate`  | Generate TypeScript/React code following standards | `/generate DataTable component with sorting`  |
| `/review`    | Review code for TypeScript/React standards compliance | `/review src/components/Button.tsx`      |
| `/refactor`  | Refactor code to align with TypeScript/React standards | `/refactor src/hooks/useAuth.ts`         |
